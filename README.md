## 📌 Overview

This project analyzes depression-related patterns using text, image, and audio data.

The implementation includes:

* NLP-based text processing
* CNN-based image analysis
* Audio feature extraction using Librosa

The project explores machine learning and deep learning techniques across multiple data modalities.
